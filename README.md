# splunk-log-analysis-capstone
SOC Capstone Project – Incident Investigation Using Splunk
 SOC Capstone – Splunk Incident Investigation

This repository documents a simulated SOC investigation conducted using Splunk.

## Project Overview
The project analyzes web, authentication, and system logs to reconstruct a full cyberattack lifecycle, including reconnaissance, initial access, persistence, lateral movement, and data exfiltration.

## Key Activities
- SQL Injection detection
- SSH brute force analysis
- Base64 reverse shell decoding
- Lateral movement detection using Nmap
- Data exfiltration analysis

## Tools Used
- Splunk Enterprise
- CyberChef

## Skills Demonstrated
- Log ingestion and indexing
- SPL query development
- SIEM investigation
- Incident response analysis
- Technical reporting

> This project is a simulated SOC lab created for educational and portfolio purposes.
